#  Barkod Destekli Kütüphane Takip Sistemi

Bu proje, küçük ve orta ölçekli kütüphaneler için geliştirilmiş, barkod okuma ve otomatik kitap bilgi çekme özelliklerine sahip bir yönetim yazılımıdır.

##  Özellikler
**Barkod Tarama:** Kamera üzerinden anlık barkod okuma.
**Otomatik Bilgi:** Google Books API ile kitap adı ve yazar bilgilerini otomatik çekme. 
**Excel Raporlama:** Tüm listeyi tek tıkla CSV/Excel formatında dışa aktarma.
**Resmi Arayüz:** Sade ve profesyonel kurumsal tasarım.
**Emanet Takibi:** Ödünç verilen kitapların ve gecikenlerin takibi.

## Kurulum
1. Bu depoyu indirin.
2. Gerekli kütüphaneleri yükleyin: `pip install -r requirements.txt`
3. Programı başlatın: `python main.py`
